```rust

// [T]HE [G]AME [O]F [L]IFE (in rust)

/* Keybinds: */
[K]        // Kill a random sampling of cells
[R]        // Randomize
[P]        // Toggle Pause
[SPACE]    // Pause

/* Mouse support */
[Click]   // Click a dead cell to create life
[Drag]    // Drag your mouse to create life
```
```bash
> git clone https://github.com/twhiting/tgol.rust.git
> cd tgol.rust
> cargo run
```
